# Fetp

Essa aplicação de teste foi feita para realizar o cadastro de usuários e login.


## 🚀 Rodar a aplicação

* Instale as dependências:
>    $ npm install
* Rode o projeto em ambiente de desenvolvimento:
>    $ npm start
* Faça o build do projeto
>    $ npm build
